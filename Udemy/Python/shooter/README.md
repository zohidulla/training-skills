# Pygame Project

Pygame is a set of Python[http://www.python.org] modules designed for writing games. It is written on top of the excellent SDL[https://www.libsdl.org/] library. This allows you to create fully featured games and multimedia programs in the python language. Pygame is highly portable and runs on nearly every platform and operating system.

For a nice introduction to pygame, examine the line-by-line chimp tutorial[https://www.pygame.org/docs/tut/chimp/ChimpLineByLine.html], and the introduction for python programmers[http://www.pygame.org/docs/tut/intro/intro.html].

More information:

Readme[http://www.pygame.org/readme.html] - Pygame Readme Information
Install[http://www.pygame.org/install.html] - Install Information

## Prerequisites

- Python 3.11
- pipenv (install using `pip install pipenv`)

## Create .venv folder

`mkdir .venv`

## Install dependencies

`pipenv install`

## Activate virtual environment

`pipenv shell`

## Run shooter game

`python main.py`

## Run rectangle game

`python rectangle.py`
